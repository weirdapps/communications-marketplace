# Comms Marketplace

Claude Code plugins for communications, presentations, and corporate branding.

## Plugins

### nbg-presentation-format
Create presentations and visual assets following National Bank of Greece (NBG) corporate brand guidelines.

**Features:**
- Complete slide specifications (12.192" x 6.858" custom dimensions)
- NBG color palette with usage guidance
- Typography specs (Aptos font family)
- Icon generation guidelines (solid fill, monochrome SVG)
- Chart configurations and color sequences
- Slide layout catalog with positioning references

**Triggers:** "NBG presentation", "NBG format", "National Bank of Greece"

## Assets

The `assets/` directory contains:
- `nbg-logo-gr.svg` - Greek NBG logo (214 x 62 px)
- `nbg-logo.svg` - English NBG logo (294 x 62 px)
- `nbg-logo-fallback.png` - PNG fallback
- `NBG-PRESENTATION-SPEC.md` - Complete specification document

## Installation

Clone this repository to your Claude Code plugins directory:
```bash
cd ~/.claude/plugins/marketplaces/
git clone https://github.com/weirdapps/comms-marketplace.git
```
