<objective>
Create infographics following NBG brand guidelines using the manage-nano-banana skill for image generation.

User request: $ARGUMENTS
</objective>

<process>
Rules for NBG infographics creation:

1. **Primary purpose**: Explain content directly and easily
2. **Style**: Clean, modern, professional - not fancy
3. **Colors**:
   - Primary accent: #007B85 (NBG Teal) - for highlights, lines, design details
   - Secondary accent: #00DFF8 (Bright Cyan) - for emphasis
   - Text: #003841 (Dark Teal) for headings, #202020 for body
   - Background: White (#FFFFFF) or Transparent
4. **Typography**: Use Roboto or Aptos font
5. **Orientation**: Landscape unless user specifies otherwise
6. **Aspect ratio**: 16:9 preferred

Follow NBG brand constraints exactly, but design content creatively.
</process>

<success_criteria>
- Infographic generated successfully
- Colors match NBG brand palette
- Clean, professional appearance
- Content is clear and informative
</success_criteria>
