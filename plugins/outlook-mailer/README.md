# Outlook Mailer

Send emails via Microsoft Outlook on macOS using AppleScript.

## Usage

```
/send-mail send an email to user@email.com with subject "Hello" and attach report.xlsx
```

## Features

- Send or save as draft
- HTML-formatted body (Aptos 12pt)
- To, CC, BCC recipients (auto-CCs sender)
- File attachments
- Outlook signature added automatically

## Requirements

- macOS with Microsoft Outlook installed
- Outlook account configured
